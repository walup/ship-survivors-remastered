# Ship Survivors
I am adding to the AI slop by using Claude to remaster a game I made about 10 years ago with Java. 
It can be played here

<https://walup.github.io/ship-survivors-remastered>

Everything about the game sucks, including the drawings, and the story made from combining song lyrics lol, but it's a good memory. I made the songs, since I had recently acquired a MIDI cable for the piano. There were many symbolisms from my experiences of that time. The arduino gun attached to a heart came from the EKG we were designing/building for a school project. I also felt like some of the ''magic'' of learning was being depleted by the homework, which consumed time I would normally spend exploring physics or biological concepts on my own. This partly motivated the story. There are other references there which I no longer remember.
